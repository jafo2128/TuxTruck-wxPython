Well, at this point, I don't have a lot to put here. But I have some general thoughts, and a rough plan of the timeline.

Upcoming Work

Here are my plans for the near future:

  1. Develop UI for audio playing
  2. Develop basic playlist handling functionality, test with a sample (fixed file) playlist.
  3. Figure out how to search music, how to hold results
  4. Develop a playlist chooser and search functionality, with the AudioPlayer_Menu panel
  5. Test playlist functionality and searching
  6. Write a script to build search indexes and playlists, run, test
  7. Load playlists, test full audio functionality
  8. Start work on navigation and some hardware (radio? phone? temperature?)

General Milestone Goals

2008-05-25 - Finish audio playing and playlists, related UI
2008-06-10 - Figure out navigation, test implementation
2008-07-01 - Finish major software components, release alpha version
2008-07-25 - Finalize major software components, order hardware
2008-08-15 - Working basic (audio only) install on final hardware.
2008-08-30 - Beta release, nearly full functionality on final hardware.
2008-09-07 - Hardware permanently installed in my truck, nearly complete software. Final testing.
