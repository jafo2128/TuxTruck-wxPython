Planned Features

Jump to: navigation, search

Some of the planned features:

  • Connectivity via Wi-Fi (802.11) and possibly Verizon EVDO AirCard (or EVDO via phone/bluetooth)
  • Realtime GPS location
    • GPS mapping, turn-by-turn directions, possibly text-to-speech
  • Handsfree car-kit for cell phone (via bluetooth)
    • automatic radio muting, obviously
    • phonebook download-upload
    • contact searching, favorite contacts
    • Caller ID names, pictures, custom ringtones
    • Blocked caller List
    • Ignore call, reply with default text (such as "I'm driving right now, will call you back soon.")
  • Audio output to amplifier (to factory speakers)
    • FM (and possibly AM) radio tuner
    • Streaming internet radio
    • Play MP3s, playlists, sort and search by artist, title, album, genre, etc.
    • Playlist by above criteria
    • Hard volume and mute controls?
    • Podcasts - download via WiFi, build daily playlist
  • Live weather data (AirCard or cached when WiFi available)
  • Heads-up-display feature
    • External Temperature
    • Time
    • Speed
    • RPMs, fuel efficiency
  • Light sensor / photoresistor for automatic day/night mode
  • OBD-II
    • Realtime performance data
    • Fuel mileage calculation in realtime
    • Error/diagnostic message view
  • Smart connection manager - prompt to use WiFi (if Available), ask whether to use AirCard or phone data


Possible additional features:

  • Video player
  • Stream TV from MythTV
  • Cell phone handsfree / voice dial
  • Control via bluetooth keypad mounted on steering wheel (4-way, center button, mute, volume controls)
  • Voice commands - GPS, radio, mute, call, etc.
  • Incoming SMS display
  • Simple web browsing?
  • Simple email?
  • Simple SMS compose?
  • Speech recognition for web browsing, email, SMS, etc.?
